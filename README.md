# WeatherBoard

WeatherBoard é um projeto focado em mostra um clima da cidade pesquisada via API ([WeatherAPI](https://openweathermap.org/api)).

## Recursos adicionados

**Versão Demo 0.5:**

1. Painel do clima da cidade pesquisada
1. Pesquisa atráves de uma caixa de texto
1. Uso de uma API para melhores informações
1. Trocador de Tema Claro/Escuro
1. Responsividade para vários aparelhos
